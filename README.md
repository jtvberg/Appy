# Appy - MacOS App Launcher

Because the Tahoe one sucks.

## Features

- Sits in the status menu.
- View as icons, icons with labels or as a list.
- Show all apps and sort alphabetically or by newest.
- You can group apps into named folders or show apps grouped by application category.
- Add items to groups by dragging or via right-click menu.
- Change the size of the items in either icon or list mode.
- Hide items you don't want to see.
